About

Who I am?

My name is Ezequiel, but you already know that because of my nick :)

I started in computer security when I was a teenager, reading ezines and “hacking” forums that led me to learn the basics of computer and networks.

From that, I dedicated myself to learn everything about computer security, playing CTFs, solving hacking challenges, reading each published research, etc.

A self-taught, passionate about what I do, and committed to improve day by day.

Professionally, I have extensive experience working as a full stack developer and security consultant performing code review, internal/external pentest and cloud security assessments.

Speaker at several conferences, trainings and workshops: OWASP Argentina, OWASP Uruguay, CharruaCon, 8.8. Chile, Ekoparty, Security Jam, private companies, etc.

OSCP (Offensive Security Certified Professional) certified.
